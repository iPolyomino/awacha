awacha

simple note web application

You can send and receive text from command line.

Demo


awacha.com/

Usage

$ curl -X POST -d "hello world" awacha.com/helloworld

-> Created

$ curl awacha.com/helloworld

-> hello world

$ curl -X DELETE awacha.com/helloworld

-> Deleted
